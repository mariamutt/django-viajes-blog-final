Acerca de MiBlog

Este blog es el proyecto final de la cursada de Python/Django. Fue diseñado para cumplir con todos los requisitos funcionales, incluyendo un sistema robusto de autenticación, una aplicación principal con CRUD completo y un módulo de mensajería privado.

Características Clave Implementadas

  • **Estructura Modular:** Separación de lógica en las apps `travels`, `accounts` e `inbox`.
  • **CRUD Completo:** Vistas para Crear, Leer, Actualizar y Eliminar Artículos de Viaje.
  • **Autenticación Segura:** Manejo de Login, Logout y Registro.
  • **Herencia de Templates:** Uso de `base.html` para un diseño consistente (NavBar y Footer).
  • **Diseño Responsivo:** Implementación de Bootstrap para una experiencia móvil amigable.
